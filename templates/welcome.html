Health Statistics Tracker

Thank you for visiting this website. If you would like to use the features, please either sign up, if you do not have an account, or log in.

This web app can allow users to enter health statistics that they would like to keep track of. Users can input different statistics and check how what their statistics look like over a long period of time. This is meant to promote healthy living and habits, since users can check on their own personal health via this web app. It's as easy as signing up or logging in!
